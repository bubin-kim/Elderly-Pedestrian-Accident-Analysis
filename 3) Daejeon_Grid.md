## Daejeon_Grid
### QGIS 활용
대전시 경계 데이터 불러오기 -> 육각형 격자 생성 -> 경계에 맞게 Clip 수행 -> 최종 결과 생성  

대전시 내 고령 보행자 교통사고를 공간적으로 분석하기 위한 기초 작업으로, QGIS를 활용하여 연구 대상 지역을 육각형 격자 형태로 변환하였다.  
-> 기존 행정구역 단위 분석의 한계를 보완하고, 보다 균일한 공간 단위에서 사고 분포를 파악하기 위한 목적을 가진다.  

고령 보행자 교통사고의 공간적 분포 패턴을 파악하고 특정 지역에 사고가 집중되는 핫스팟 존재 여부를 확인을 해볼 예정이다.

사각형 격자와 비교했을 때, 육각형 격자는 인접 셀 간 거리 편차가 적고  
중심에서 주변까지의 거리 왜곡이 적어 공간 분석에서 보다 안정적인 결과를 제공한다.  
따라서 교통사고와 같은 공간적 밀집 현상을 분석하는 데 적합하다고 판단하였다.
